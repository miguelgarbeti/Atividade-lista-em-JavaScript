<h1 align="center">📋​​ Os 60 Exercícios de Lógica</h1>

O objetivo é fazer 60 atividades de 3 listas separadas por complexidade é desenvolvendo raciocínio lógico e consolidar conhecimentos em JavaScript através da prática com exercícios clássicos.
<h1 align="center"> 📂 Divisão das Listas:</h1>

## Ordem de complexidade das Pastas:

 [📏 01_EstruturaSequencial (~20 exercícios): * Foco: Uso de let , operadores matemáticos e Number(prompt())](Pasta_1)
---------------
 [🔁​ 02_EstruturaDeDecisao (~20 exercícios): * Foco: Substituir o if/else do C pela sintaxe JS.](Pasta_1)
---------------
[⚙️​ 03_EstruturaDeRepeticao (~20 exercícios): * Foco: Implementação de laços while, do...while e for.](Pasta_1)
---------------
 ## 🚀 Como executar.

1. Instale o Node.js

2. Acesse a pasta do projeto:
   cd Pasta de desino

3. Execute um exercício:
   node ex_numero-do-arquivo.js
